🚀 TaskFlow App

A modern Task Management Web App built using React, Firebase, and Tailwind CSS.
It helps users manage daily tasks efficiently with authentication, filtering, and calendar view.

✨ Features
🔐 User Authentication (Login / Signup / Logout)
📋 Create, Edit, Delete Tasks
🔍 Search & Filter Tasks
📅 Calendar View for task scheduling
📊 Dashboard overview
👤 User Profile & Settings
⚡ Real-time data using Firebase Firestore
🛠️ Tech Stack
Frontend: React (Vite)
Styling: Tailwind CSS
Backend: Firebase
Database: Firestore
Authentication: Firebase Auth
Routing: React Router
